# Whatsapp Puppet

Whatsapp automated send message to phone number

## Installation

git clone project

```
git clone git@github.com:fajaraditio/wapuppet.git
```

``` install necessary package
npm install
```

## How to use

Go with command
```
node wapuppet.js
```
